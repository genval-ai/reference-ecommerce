# Bluestone PIM
先進的な組み込み型PIM。製品情報を効率的に一元化するためのエンタープライズPIMソリューションで、比類のない効率性を実現します。

**プロバイダーメタデータ**
| プロパティ | 値 |
|----------|------|
| 機能コード | `bluestone-pim` |

## プロバイダー接続タイプ

### Bluestone PIM
API資格情報を使用してBluestone PIMに接続

**接続タイプメタデータ**
| プロパティ | 値|
|----------|------|
| 接続タイプコード | `bluestone-pim` |

#### APIキー
Bluestone PIMでの認証に使用するAPIキー

**接続プロパティメタデータ**
| プロパティ | 値|
|----------|------|
| プロパティコード | `api_key` |
| 機密情報 | はい |
| プロパティレベル | クライアント |
| 必須 | はい |
| 例値 | bstn_1234567890abcdef |

#### APIベースURL
Bluestone PIM APIエンドポイントのベースURL

**接続プロパティメタデータ**
| プロパティ | 値|
|----------|------|
| プロパティコード | `base_url` |
| 機密情報 | いいえ |
| プロパティレベル | クライアント |
| 必須 | はい |
| 例値 | https://api.bluestone.com |
